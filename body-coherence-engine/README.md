# The Body as a Coherence Engine

**Governing question:** How can the brain affect the body not only by commanding muscles, but by changing the mechanical field in which muscles, fascia, gravity, and sensation cohere?

## Summary

The brain has two strategies for changing the body: direct muscle command (the familiar neuromuscular pathway) and mechanical repositioning (changing the body's arrangement in gravity so that load, sensation, and tone redistribute). The second strategy is mapped onto CNOS coherence architecture — the body closes model-reality gaps the same way a coherent agent does: observe, act, compare, revise.

## Key concepts

- **Command vs configuration** — squeezing a muscle harder vs placing the body so the effect emerges
- **String-instrument metaphor** — muscle as tuner, bone as bridge, fascia as resonant field, gravity as the constant
- **Mechanotransduction** — chemistry produces force, force produces chemistry; the distinction is real but not a separation
- **CNOS mapping** — CMP → MCP → CAP → CLP applied to body-brain coherence loop
